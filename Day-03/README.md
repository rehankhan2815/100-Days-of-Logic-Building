# Day 3

## Problem 8

* **Concept:** Average of Positive Numbers
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Some problems combine multiple patterns. This problem combines counting and accumulation (sum) to calculate an average.

## Problem 9

* **Concept:** First Positive Number
* **Time Complexity:** O(n) (Worst Case)
* **Space Complexity:** O(1)
* **Lesson Learned:** Use the Early Return pattern. Return immediately when the required element is found, otherwise return `None` after the loop finishes.

## Problem 10

* **Concept:** Count Occurrences of a Target
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Many problems are the same template with only the condition changing. This is the same counting pattern used earlier, with a different condition.

## Bonus Learning

* **Concept:** Pattern Recognition
* **Lesson Learned:** Before writing code, identify the underlying pattern (Tracking, Counting, Accumulation, or Early Return). Solving by recognizing templates is more effective than treating every problem as new.
