## Problem 25

* **Concept:** Second Largest Distinct Element
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Two-tracker problems require maintaining the relationship between the largest and second largest elements while ignoring duplicates.

## Problem 26

* **Concept:** Difference Between Largest and Second Largest Distinct Elements
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Existing algorithms can often be reused with only small modifications instead of writing a new solution from scratch.

## Problem 27

* **Concept:** Second Largest Distinct Even Number
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Complex problems can often be solved by combining a filtering condition with an existing tracking pattern.

## Bonus Learning

* **Concept:** Code Reusability & Operator Precedence
* **Lesson Learned:** Reusing proven logic leads to cleaner solutions. Also, using `elif` and proper parentheses improves readability and prevents subtle logical bugs.
