# Day 2

## Problem 4

- **Concept:** Counting Positive Numbers
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
- **Lesson Learned:** Counting problems use a counter initialized to 0. The pattern stays the same; only the condition changes.

## Problem 5

- **Concept:** Count Even and Odd Numbers
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
- **Lesson Learned:** Multiple counters can be maintained in a single traversal, and meaningful variable names improve code readability.

## Problem 6

- **Concept:** Sum of Positive Numbers
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
- **Lesson Learned:** Accumulation follows the same pattern as counting; instead of adding 1, add the value itself.

## Bonus Learning

- **Concept:** Largest Even Number (Sentinel Values)
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
- **Lesson Learned:** Use `None` as a sentinel value to represent "no valid answer found yet" and update it when the first valid element is encountered.
