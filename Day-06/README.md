# Day 6
## Problem 18
* **Concept:** Count Multiples of 3
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Counting problems follow the same template; only the condition changes.
## Problem 19
* **Concept:** Count Positive Even Numbers
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Multiple conditions can be combined using logical operators (and) while keeping the same counting pattern.
## Problem 20
* **Concept:** Largest Positive Number
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Using None as a sentinel value makes tracking valid elements cleaner than relying on infinity values when filtering is involved.
## Problem 21
* **Concept:** Smallest Odd Number
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:**Tracking with None works well when searching for the minimum or maximum element that satisfies a specific condition.
## Bonus Learning
* **Concept:** Tracking + Filtering
* **Lesson Learned:** A tracking problem becomes more powerful when combined with a filtering condition. First filter the required elements, then update the tracking variable only for valid candidates.
