# Day 7

## Problem 22

* **Concept:** Second Smallest Distinct Element
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Maintaining two tracking variables requires careful handling of duplicates. The second smallest element must be distinct from the smallest.

## Problem 23

* **Concept:** Largest Negative Number
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Tracking problems can be combined with filtering conditions by updating the tracking variable only for valid elements.

## Problem 24

* **Concept:** Count Numbers Divisible by Both 2 and 5
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** A mathematical observation can simplify conditions. Checking divisibility by both 2 and 5 is equivalent to checking divisibility by 10.

## Bonus Learning

* **Concept:** Two-Tracker Pattern
* **Lesson Learned:** Not every tracking problem is a simple variation of finding a maximum or minimum. Some require maintaining a relationship between two variables while correctly handling edge cases and duplicates.
