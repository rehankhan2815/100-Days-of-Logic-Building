# Day 5

## Problem 14

* **Concept:** Count Numbers Greater Than a Target
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Many counting problems differ only in the condition inside the `if` statement.

## Problem 15

* **Concept:** Sum of Even Numbers
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Accumulation problems follow the same template; only the condition changes.

## Problem 16

* **Concept:** Difference Between Largest and Smallest Element
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Two tracking variables can be maintained simultaneously in a single traversal. Initialize them outside the loop using sentinel values (`-inf` and `inf`).

## Problem 17

* **Concept:** Count Numbers Within a Range
* **Time Complexity:** O(n)
* **Space Complexity:** O(1)
* **Lesson Learned:** Python's chained comparison (`low <= num <= high`) makes range checks clean and readable.

## Bonus Learning

* **Concept:** Pattern Recognition
* **Lesson Learned:** By Day 5, similar problems are becoming recognizable by their underlying pattern instead of their wording. The focus is shifting from "How do I solve this?" to "Which pattern is this?"
